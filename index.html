Selenium WebDriver is an open-source automation framework for web browsers. It allows you to write tests in Java, Python, C#, and other languages to interact with web elements.
Playwright is a modern end-to-end testing framework by Microsoft. It supports Chromium, Firefox, and WebKit and provides auto-wait capabilities, intercepting network requests, and more.
REST API testing validates HTTP endpoints by sending GET, POST, PUT, DELETE requests and verifying responses. Tools include REST Assured (Java), Axios, and Supertest.
POM is a design pattern that creates an object repository for web elements. Each page is represented as a class with elements as fields and actions as methods.
